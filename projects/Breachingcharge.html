Breachingcharge - DayZ Mod

  • Github
  • Steam
  • Wiki

This mod adds a configurable base raiding system to the game DayZ. Building your own base and raiding other's is a big part of the game and the main focus of many players, especially on modded servers and during the later stages of a player's progression.

While this was my first public mod for the game and there are many things that could be improved (many of which have been implemented in my newer mod More Explosives), it still managed to garner a significant user-base of over 1.7 million users since it was first released.
