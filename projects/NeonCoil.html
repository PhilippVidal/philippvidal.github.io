Neon Coil

  • Github
  • Release

Neon Coil offers familiar gameplay with a modern twist on the classic games Snake and Space Invaders. It is a small game created within a timespan of 2 weeks to become more accustomed with some of Unreal Engine's newer features, such as Niagara, Lumen and the Enhanced Input System, while also exploring the possibilites offered when creating 2D games using Paper2D.

My work on this game includes a custom ability system (similar to the Gameplay Ability System), the associated abilities (e.g., Sprinting, Shooting, etc.), and the general input & controls for the player using Unreal's Enhanced Input System.
Additionally I created the different Niagara effects and custom materials for game to achieve a stylized appearance.
